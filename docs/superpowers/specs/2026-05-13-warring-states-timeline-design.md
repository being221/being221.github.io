# 春秋战国历史时间线 — 设计文档

## 概述

在个人主页中新增独立页面 `timeline.html`，以垂直左脊线时间线展示春秋战国的 ~150+ 条历史事件，支持分类筛选、搜索、点击展开详情和因果关系可视化。

## 架构

单文件 `timeline.html`，包含 HTML 结构 + CSS 样式 + JS 逻辑，无外部依赖。事件数据以 JS 数组内联。

## 文件

| 文件 | 职责 |
|------|------|
| `timeline.html` | 全部代码：页面结构、竹简风样式、事件数据、交互逻辑 |
| `index.html` | 微小改动：顶部导航加"春秋战国"链接 |

## 数据模型

```js
const events = [
  {
    id: 1,
    year: -771,            // 负数=公元前
    title: "犬戎攻破镐京",
    country: "周",
    category: "战争",
    summary: "犬戎攻破镐京，杀周幽王，西周灭亡。",
    detail: "周幽王宠褒姒，废申后与太子宜臼……（详细叙述）",
    causes: [],            // 前因事件 ID 列表
    effects: [2, 3],       // 后果事件 ID 列表
  },
  // ... 约150+条
];
```

### 分类枚举

**国家（15个）：** 周、齐、楚、燕、韩、赵、魏、秦、吴、越、鲁、宋、郑、卫、其他

**类型（8个）：** 战争、会盟、改革、人物生卒、思想著作、政变、外交、其他

## 页面布局

```
┌─────────────────────────────────┐
│  [搜索框]  [筛选按钮]            │  ← 顶部简洁条，滚动时吸附
├─────────────────────────────────┤
│                                 │
│  ●──── 前771                     │
│  │    犬戎攻破镐京               │  ← 左脊线 + 事件卡片
│  │    西周灭亡                   │     点击展开详情
│  │                              │
│  ●──── 前770                     │
│  │    平王东迁                   │
│  │                              │
│  ●──── 前685                     │
│  │    齐桓公即位                 │
│                                 │
└─────────────────────────────────┘
```

- 左脊线：竖线居中偏左（屏幕约 15% 处），墨绿色
- 事件圆点：位于脊线上，颜色按类型区分
- 卡片：圆点右侧，浅米色底，显示年份+标题+摘要，点击展开详情
- 因果连线：有关联的事件之间以淡色箭头弧线连接（视觉上不干扰阅读）

## 交互

### 展开/收起
点击卡片展开详情区——显示完整叙述 + 前因/后果事件链接

### 筛选
点击"筛选"按钮弹出面板：
- 国家复选框（默认全选）
- 类型复选框（默认全选）
- 时间范围滑块（公元前 770 ~ 前 221）
- "应用"按钮关闭面板并刷新时间线

### 搜索
搜索框实时过滤，匹配标题和摘要中的关键词

### 因果连线
- 时间线上：相关事件间画出淡色箭头弧线
- 卡片详情内：列出"前因"和"后果"链接，点击跳转到对应事件

## 视觉风格（竹简风）

| 元素 | 配色 |
|------|------|
| 页面背景 | `#f5f0e8` 浅米色 |
| 卡片背景 | `#faf7f0` 略带暖白 |
| 脊线 | `#5b8c5a` 墨绿 |
| 文字主色 | `#3d3226` 深褐 |
| 圆点/强调 | 按类型：战争`#c44`、会盟`#5b8c5a`、改革`#d4a017`、人物`#6b7b8d`、思想`#8b5e3c`、政变`#8844aa`、外交`#4488aa`、其他`#999` |
| 因果连线 | `#c4b998` 淡褐色半透明 |

字体：系统默认中文衬线体（`"Noto Serif SC", "Source Han Serif SC", "STSong", serif`）

## 响应式

- 桌面（>768px）：左脊线在 ~15% 位置，卡片最大宽 600px
- 手机（≤768px）：脊线在 8% 位置，卡片字体略小，搜索框和筛选按钮各占一行
