# 我的起卦应用

一个基于Vue 3的个人起卦应用，支持多种起卦方式和详细的卦象解释。

## 项目特点

- 📱 响应式设计，支持移动端
- 🎲 支持摇动感应起卦
- 📊 详细的历史记录功能
- 📝 预设问题模板
- 🌐 完整的64卦数据

## 技术栈

- Vue 3
- Vite
- Vue Router
- Vitest (测试框架)

## 测试

项目已配置完整的测试套件：

### 运行测试

```bash
# 运行所有测试
npm test

# 运行测试并生成覆盖率报告
npm run test:coverage

# 启动测试UI界面
npm run test:ui

# 代码检查
npm run lint
```
